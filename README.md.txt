##E-commerce SQL Data Analysis Project

## Objective
To analyze customer behavior and sales data using SQL and generate business insights.

## Tools Used
- MySQL Workbench
- SQL

## Database Tables
- Customers
- Products
- Orders
- Order_Items
- Payments

## Key Analysis Performed
- Total revenue calculation
- Top customers identification
- Monthly sales trend analysis
- Best-selling products
- Customer retention analysis

## SQL Skills Used
- Joins (INNER JOIN)
- Aggregation (SUM, COUNT)
- GROUP BY
- Window Functions (RANK)

## Key Insights
- Top customers contribute most of the revenue
- Electronics category performs best
- Repeat customers are more valuable
- COD orders have higher cancellation rates

## How to Run This Project
1. Open MySQL Workbench
2. Run 'schema.sql'
3. Run 'data.sql'
4. Run 'queries.sql'
